# TaraNote UI v5.2 (Static Export)

A static HTML export of the Vue.js components used in the v5 production build.

## Purpose
To have a lightweight, non-framework version of the core UI flows for reference and testing.

## Files
- **[article_list.html](./article_list.html)**: Main blog home (Bento Grid).
- **[dashboard.html](./dashboard.html)**: Admin dashboard.
- **[article.html](./article.html)**: Single post view.
- **[welcome.html](./welcome.html)**: Default landing page.
